Scamper API

An API for writing SCTP clients and servers.

Examples and details are available on the homepage of the GitHub project. For use from Maven, see the instructions here.

This package uses Netty 5 and Google Guice under the hood.

It is not necessary to know much about Guice to use this library, but it is handy for binding configuration your objects will need. Your MessageHandler implementations will need a constructor annotated with @Inject or a no-argument constructor.

The main entry point is SctpServerAndClientBuilder; to use, create some MessageTypes and pass them to the builder's bind() method along with your implementation of MessageHandler which will receive and process messages of those types. Then use one of the build() methods to get a server or client you can start().
